Instructions:

Below you are prompted with different websites you can take a usability test for. Once you first load whichever website you choose you will notice that it is all "jumbled" up. You will then begin taking a timed usability test for this website. Upon completion of this initial usability test you will then load the page to its normal state and be prompted to take the same usability test. To begin, select a website below and hit the green button. If at any point you run into any issues refresh the page and start over. (Note: to bypass cors I use an API made by herokuapp that enables crossorigin requests. Because of this, the content might take a minute to load or ultimately fail to load every once in a while.)
